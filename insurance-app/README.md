# 내 보험 한눈에 📱🛡️

내가 가입한 **보험 회사·상품**과 **보장 범위**를 한눈에 정리하고,
"이런 일이 생겼을 때 내 보험으로 보험금을 받을 수 있나?"를 쉽게 알려주는 **개인 보험 정리 앱**입니다.

> 보험 용어가 어렵고, 막상 일이 생겼을 때 내가 가진 보험으로 혜택을 받을 수 있는지 모르는 사람을 위해 만들었습니다.

## ✨ 핵심 기능

| 탭 | 하는 일 |
|----|---------|
| 📋 **내 보험** | 회사·상품·종류·보험료·갱신여부·보장내용을 카드로 정리 (추가/수정/삭제) |
| 🛡️ **보장보기** | 흩어진 보장을 **카테고리별**(진단·입원·수술·실손·사망 등)로 묶어서 한눈에 |
| 🔎 **상황별** | "암 진단", "입원", "교통사고" 등 **상황을 고르면** 내 보험 중 해당 보장 + 청구 방법 안내 |
| 📖 **용어사전** | 실손·면책기간·자기부담금·갱신형 등 어려운 말을 쉬운 한국어로 풀이 (검색 가능) |

### 추가 기능
- ⏰ **갱신일·만기 알림** — 만기일/갱신일을 넣으면 "다가오는 일정"으로 미리 보여주고, 갱신주기를 고르면 다음 갱신일을 자동 추정합니다. 알림 권한을 켜면 7일 이내 임박 시 앱을 열 때 알림도 보냅니다.
- 📷 **증권 사진 첨부 + OCR 자동입력** — 증권/보장내용 화면을 찍어 보관하고, 사진에서 글자를 읽어 회사명·상품명·증권번호를 자동으로 채워줍니다(인터넷 연결 필요, [Tesseract.js](https://tesseract.projectnaptha.com/) 사용).
- 🔬 **중복 보장·보장 공백 분석** — "보장보기" 탭 상단에서 여러 보험이 겹치는 보장(특히 실손 중복)과, 일반적으로 권장되는데 빠진 보장을 짚어줍니다.
- ⌨️ **입력 편의** — 보험사 자동완성, 종류별 **보장 템플릿(🧩)** 한 번에 채우기, 금액 입력 시 "만원/억원" 미리보기.
- 👨‍👩‍👧 **가족 구성원별 관리** — 피보험자(본인·배우자·자녀…)별 필터와 요약, 정렬(갱신 임박순·보험료순 등).
- 🧾 **보험금 청구 도우미** — 상황별 화면에서 "청구 체크리스트"를 만들어 서류 준비/접수/지급 진행상황을 체크·추적. 진행 중인 청구는 홈 상단에 표시.
- 📥 **보험 목록 한번에 가져오기(반자동 등록)** — [내보험찾아줌](https://cont.insure.or.kr/)/보험사 앱의 보유계약 목록을 복사해 붙여넣으면 여러 건을 한꺼번에 분석·등록.
  > ℹ️ 진짜 "자동 조회"는 마이데이터 사업자(토스·뱅크샐러드 등)만 가능합니다. 이 앱은 폰에만 저장되는 개인 도구라, 본인이 목록을 붙여넣는 반자동 방식으로 제공합니다.

- 데이터는 **내 폰(브라우저)에만 저장**됩니다(보험 정보 localStorage, 증권 사진 IndexedDB). 서버로 전송하지 않습니다.
- **백업 내보내기/불러오기**(JSON) 지원 — 기기 변경 대비. (사진은 용량 문제로 백업에 미포함)
- **PWA**라서 홈 화면에 설치하면 일반 앱처럼 쓰고, **오프라인에서도** 열립니다. (OCR만 인터넷 필요)

## 🚀 폰에서 바로 쓰는 법

### 방법 A. GitHub Pages로 호스팅 (추천)
1. 저장소 **Settings → Pages**에서 브랜치를 `main`(또는 현재 브랜치), 폴더를 `/ (root)`로 설정해 배포합니다.
2. 발급된 주소 뒤에 `/insurance-app/` 를 붙여 폰 브라우저로 엽니다.
   예: `https://<아이디>.github.io/<저장소>/insurance-app/`
3. 크롬/사파리 메뉴에서 **"홈 화면에 추가"** → 앱처럼 사용.

### 방법 B. 내 컴퓨터에서 바로 확인
```bash
cd insurance-app
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080 접속
```
> ※ 서비스워커(오프라인 기능)는 `file://`로 열면 동작하지 않으니, 위처럼 로컬 서버로 여세요.

## 🗂️ 파일 구성
```
insurance-app/
├─ index.html      # 화면 뼈대
├─ styles.css      # 모바일 UI 스타일
├─ app.js          # 화면 렌더링 · 저장 · 상황 매칭 로직
├─ data.js         # 보장 카테고리 / 상황별 안내 / 용어사전 / 예시 데이터
├─ manifest.webmanifest  # PWA 설정
├─ sw.js           # 오프라인 캐시(서비스워커)
└─ icon*.svg       # 앱 아이콘
```

## ⚠️ 중요 안내
이 앱은 **개인이 보험을 정리·이해하기 위한 도구**입니다.
상황별 안내와 용어 설명은 **일반적인 내용**이며, 보험 상품은 회사·약관마다 다릅니다.
**실제 보장 여부와 보험금 액수는 가입한 보험의 약관·증권과 보험사 심사**로 결정됩니다.
정확한 내용은 보험사 앱의 "보장내용 조회"나 콜센터로 확인하세요.
